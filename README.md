# Japonica

[![Netlify Status](https://api.netlify.com/api/v1/badges/e3556637-32c5-438c-b5ff-2b361ac3ebdc/deploy-status)](https://app.netlify.com/sites/japonica/deploys)

A blog system with Elm, siteelm, and Netlify CMS.

## Setup

```
npm i
```

## Run dev server

```
npm run dev
```

## Production build

```
npm run build
```

Then, see `/dist`.
